# Directory Listing — Promotional Assets

Promotional cards for the Claude MCP Directory listing of Git Repo Auth.
Each card is composed as a "certificate of minting" and shows **real tool
output** captured during pre-submission validation against production
(v0.3.0): the bare read-only mint, GitHub's 403 refusing a push with a
read token, the cached free re-mint, and the kill-switch wall list.

- `card-1.png` … `card-4.png` — 1600×1000 PNG, brand palette
  (pine `#08322A`, viridian `#0E5A4A`, paper `#FAFAF6`, amber `#D9A441`),
  Archivo + IBM Plex Mono.
- `paired-prompts.txt` — the prompt that produces each card's output.
- `cards.html` — the source; render via headless Chromium at 1600×1000.
  The seal image is the deployed `/icon-512.png`.
